Пётр Семёнович ПГК:
-Теперь, когда мы прошлись по основным настройкам, пора сформировать твою первую грузовую базу у нас, чтобы выполнить на ней расчет. -Уверен, ты умеешь работать с Excel'ем. Итак, скачивай шаблон для заполнения здесь Шаблоны грузовых баз -Запомни эти простые, но важные правила заполнения шаблона. Вся компания пользуется ими:
  • Использовать нужно полную грузовую базу, т.е. базу, которая содержит все перевозки, которые могут быть погружены парком ПГК
  • Если твоя база содержит корреспонденции по каждому вагону, ее нужно преобразовать, т.е. сделать сводную таблицу
  • Когда будешь делать сводную таблицу, проверь, что количество вагонов и ставка имеют верные значения, т.е количество вагонов - это количество, а не сумма;
  • Для заполнения столбца "Обязательные перевозки" тебе нужно определить в своей грузовой базе min, которые будут обозначаться цифрой 1-значит обязательные перевозки, max, которые будут обозначаться цифрой 0-зачит не обязательные перевозки
  • Поля не должны содержать знаков и символов, например # , #ДЕЛ/0! и прочих
  • Минимальная ставка может быть 0,01руб и максимальная 500 000руб
  • Количество вагонов не должно быть равно 0 или быть пустым полем
  • Коды станций отправлений/назначений должны быть 6-ти значными, не должно быть пустых полей
  • ID клиента должен соответствовать договору клиента в общекорпоративном справочнике, а если ID клиента отсутствует, нужно поставить 0
  • В полях примечание можно писать нужные для себя данные, которые пригодятся для дочерних расчетов
